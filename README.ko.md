# 👋 소개
안녕하세요 — 제 이름은 **이종효**이며, **Bluehill**로도 활동합니다.

저는 취미로 소프트웨어 개발을 하는 사람입니다. 주로 필요해서 직접 만드는 도구나 라이브러리를 만듭니다.

## 🔭 현재
- 개인 프로젝트를 진행하고 새 기술을 학습하고 있습니다.

## 🛠 기술 및 도구
- 언어: C#, TypeScript, Kotlin (조금), Python (아주 조금)
- 도구: Git, GitHub Actions, Visual Studio (ReSharper 사용), Visual Studio Code, JetBrains IDE들
- 기타: Windows, Linux

## 📂 주요 프로젝트
- [Bluehill.Hangul](https://github.com/BluehillNuGet/Bluehill.Hangul) — .NET 한글 라이브러리 (조사 처리, 음절 분해 등)
- [Bluehill.Analyzers](https://github.com/BluehillNuGet/Bluehill.Analyzers) — Roslyn C# 분석기 및 소스 생성기
- [GitHub Pages Blazor WASM](https://github.com/na1307/blazor-github-pages) — Blazor WebAssembly 앱을 GitHub Pages에 배포할 수 있도록 준비해 주는 GitHub Action
- [Comment Remover for IntelliJ](https://github.com/na1307/intellij-comment-remover)
- [Encoding Converter](https://github.com/na1307/EncodingConverter) — 코드 파일 인코딩을 쉽게 변경할 수 있게 해주는 Visual Studio 확장 및 WinUI 3 앱
